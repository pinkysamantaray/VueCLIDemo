# VueCLIDemo
> vue ui
> vue create
>  Add npm task in package.json: 
    "serve": "vue-cli-service serve",
    "build": "vue-cli-service build"

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
